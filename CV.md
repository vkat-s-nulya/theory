## 13. Опыт работы

Институт телекоммуникаций, Программист C++: 03/2024 - настоящее время
Занимался разработкой 3D-тренажёра трёх гусеничных машин на C++/Unigine для обучения и аттестации механиков-водителей. Тренажёр состоит из трёх компонентов — Инструктора, Визуализации и Модели — связанных между собой по сети:
- Разработал на C++/Qt модуль «Инструктор» — рабочее место преподавателя для запуска тренажёра, задания параметров упражнения и типа машины, наблюдения за ходом тренировки и автоматической оценки навыков курсантов по итогам упражнения, снизив нагрузку на инструктора при аттестации
- Реализовал сетевое взаимодействие между тремя компонентами тренажёра — Инструктором, Визуализацией и Моделью — обеспечив их работу как единой системы с синхронизацией состояния в реальном времени
- Перенёс матмодель движения техники с C# на C++, сократив её с 30 000 до 7 000 строк, и покрыл модель тестами на эквивалентность
- Распараллелил расчёт физической модели движения техники на пул потоков, что устранило просадки симуляции на сценах с одновременной работой трёх машин
- Реализовал в PostgreSQL хранение упражнений, параметров машин и результатов прохождения курсантов

АМЭ, Младший программист C++: 01/2023 - 02/2024
Занимался разработкой транслятора математических схем из LabView в C++ код. Исходные модели существовали только как LabView-схемы и исполнялись лишь самой средой LabView под Windows, а заказчику требовалось запускать эти модели на Linux:
- Реализовал многопоточный парсер LabView-схем из их PNG-экспорта: находит и классифицирует блоки на изображении пулом потоков и собирает дерево схемы в JSON
- Разработал два макроса для CorelDraw: первый строит векторную диаграмму схемы по JSON, второй экспортирует её обратно в JSON — отдел ручного контроля получил возможность проверять и править распознанные схемы перед генерацией кода
- Разработал на C++ генератор кода, который по JSON собирает готовый к компиляции C++ исходник для запуска на Linux
- Покрыл схемы тестами на эквивалентность: фиксировал входы и выходы исходной модели в LabView и прогонял те же входы через сгенерированную C++ программу со сверкой выходов
